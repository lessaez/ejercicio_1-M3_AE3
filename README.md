# Ejercicio 1 – SASS patrón 7–1

Proyecto realizado como ejercicio práctico para modularizar estilos usando SASS.

## Tecnologías

- HTML5
- SASS (SCSS)
- Git y GitHub

## Estructura

Se utilizó el patrón 7–1 separando variables, mixins, estilos base y componentes.

## Compilación

```bash
npx sass assets/scss/main.scss assets/css/main.css --watch
```
